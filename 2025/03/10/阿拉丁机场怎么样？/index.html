阿拉丁机场简介

阿拉丁机场（Aladdin Network Optimizer）是2024年开业的一个性价比中转机场。Shadowsocks 协议，现在开始将逐步替换 Trojan 协议（新上了几个实验节点，0.2的倍率）。机场节点包含了几大常用地区以及一些冷门地区。

  • 机场节点：香港、日本、台湾、新加坡、美国、韩国、澳门、英国、德国、罗马尼亚、土耳其、泰国、澳大利亚、法国、荷兰、瑞典、迪拜、沙特阿拉伯、以色列、南非、越南、巴西、马来西亚、菲律宾、朝鲜、阿根廷、柬埔寨、埃及、老挝、缅甸、尼日利亚、巴基斯坦。
  • 解锁 ChatGPT 和 奈飞良好。
  • 不限速、但对同时在线的 IP 数有限制。
  • 支持支付宝和微信付款（有手续费）以及 USDT 付款。
  • 有充值活动，充值可以额外赠送一点余额。有几个等级，8% - 25%。
  • 有奈飞劫持免登录节点（一直显示 Error，我没试过）。

有个小插曲，就是在测速时，该机场的香港节点（香港1,2）的 IP 显示美国，美国节点又显示的法国。这有可能是阿拉丁机场为了解锁流媒体服务采用了 DNS 劫持或者是智能分流的策略。实际使用中，可能会造成困扰。看流媒体的时候是正常的，但在某些服务时，例如（kraken）就会造成困扰，会显示我的 IP 不在香港。暂时还不确定使用社交媒体（FB，X）会不会有这个问题（不敢试，怕封号，囧）

阿拉丁机场跑路传闻

去年7月份阿拉丁机场被 DDOS 过，运营人员流失了很多，后续是恢复了服务，给了补偿方案，后面应该运行还算稳定。我估计他们当时应该可能在 TG 上推广过。这些在机场的公告里有说。

阿拉丁机场使用感受总结

截至发文，已经用了13天。总结一下，仅代表个人感受。如果要买，月付优先，自己先试试再说。

  • 性价比高。别家同样的价格给不了这么多流量。搭配8折券，￥48能用一年。
  • 节点很多，尤其是有很多小众节点，虽然我自己用不上。
  • 目前正在转换成 Trojan 协议，0.2的倍率有点香。
  • 速度方面够用，看 YouTube 4K 我觉得卡了点（我是最低档套餐），但就这个套餐的流量也没法看 4K，不然一下就没了。
  • IP 那个问题，问了 GPT，说可能会造成 DNS 泄露。
